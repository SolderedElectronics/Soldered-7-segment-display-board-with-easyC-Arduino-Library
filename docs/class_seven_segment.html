◆ setPins()

void SevenSegment::setPins ( uint8_t _pins)

Set the pins of the 7segment display. In the byte, 0 is a segment turned on, 1 is a segment turned off.

Segment mapping: –a– | | f b | | –g– | | e c | | –d– dot (dp)

The byte is constructed as follows: Bit 7 - dot (dp) Bit 6 - segment a Bit 5 - segment b Bit 4 - segment c Bit 3 - segment d Bit 2 - segment e Bit 1 - segment f Bit 0 - segment g

Parameters
_pinsthe state of the pins, as described in the comment above
